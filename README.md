# Personal GitHub Pages Profile

This is a static academic/CV-style personal profile page ready for GitHub Pages.

## Customize

Edit `index.html` and replace:

- `Your Name`
- `Your University or Company`
- `your-username`
- `you@example.com`
- Project names, descriptions, and links
- Publication titles and links
- Awards, experience, and education entries

Replace `assets/profile-hero.png` with a headshot when you have one. Keep the
same filename or update the image path in `index.html`.

## Publish on GitHub Pages

1. Create a repository named `your-username.github.io`.
2. Add these files to the repository root.
3. Push to GitHub.
4. Open `https://your-username.github.io`.

For a project repository instead, push these files and enable GitHub Pages from the repository settings using the `main` branch root.
